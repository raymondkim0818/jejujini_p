$SYS Topic management

시스템의 집계및 현황 정보 메세지를 $SYS 토픽(topic)을 통해 지정된 시간마다 반복하여 발송합니다. 설정이동

topic을 클릭하면 상세 정보를 확인 할 수 있습니다.


# topic desc value status
1 $SYS/broker/version 서버 버전 정보 xpn version 2015.01.25.1 inactive
2 $SYS/broker/timestamp 서버 시간 2014-10-08 22:31:34+0000 active
3 $SYS/broker/uptime 서버 가동 시간 5021533 seconds inactive
3 $SYS/broker/vhosts/total Virtual host 갯수 10 active
5 $SYS/broker/clients/total 서버에 등록되어 있는 모든 클라이언트 갯수 3,242,341 active
6 $SYS/broker/clients/inactive Cliean Session으로 설정되어 있지 않고 연결이 끊긴 클라이언트 갯수 1,344,123 active
7 $SYS/broker/clients/active 현재 서버에 접속되어 있는 클라이언트 갯수 1,000 active
8 $SYS/broker/messages/stored 서버에 저장되어 있는 메세지 갯수(durable, QoS1) 1,000 inactive
9 $SYS/broker/messages/received 서버가 가동된 이후 PUBLISH 메세지를 수신(ack, QoS1)메세지 전체 갯수 1,000 inactive
10 $SYS/broker/messages/sent 서버가 가동된 이후 PUBLISH 메세지 갯수 1,000 inactive
11 $SYS/broker/bytes/received 서버가 가동된 이후 PUBLISH 메세지를 수신(ack, QoS1) 메세지 Byte 1,000 active
12 $SYS/broker/bytes/sent 서버가 가동된 이후 PUBLISH 메세지 Byte 1,000 active

$SYS topic 정보

$SYS/broker/version
서버가 가동된 이후 PUBLISH 메세지를 수신(ack, QoS1)메세지 전체 갯수
* 사용 여부를 체크 하면 다시 $SYS 필드에 추가됩니다.
