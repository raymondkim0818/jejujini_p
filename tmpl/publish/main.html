과거 발송 내역 10건의 내용을 표시합니다.
알람 Bar항목에 표시되며, 입력하지 않을경우 제목에 입력된 값으로 전달됩니다.
알람 목록의 제목 영역에 표시되며 필수 입력 항목입니다.
알람 목록의 내용 영역에 표시되며 내용 입력 제약은 없습니다.
로그인 계정에 의해 자동으로 부여되는 항목입니다. (다른 호스트측으로 메세지 발송가능)
',' 단위로 입력할경우 복수개의 메세지를 발송 할 수 있습니다.
유효기간(일단위)이 만료되면 서버에서 메세지가 삭제됩니다.
수신 대상자가 많을경우 서버 부하가 발생할수 있습니다.

* 설정값은 모든 클라이언트에게 즉시 반영되므로 명령어 종류와 값을 주의해서 설정하세요.
