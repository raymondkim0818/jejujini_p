Virtual host management

가상 호스트 (Virtual Host)기능을 통해 동시에 독립된 여러 서비스를 운용할 수 있습니다.

host을 클릭하면 상세 정보를 확인 할 수 있습니다. (todo 전체 재접속 or unack 목록 재접속)


# host (alias) created conn/limit queues ready unacknowledge total rats status # account
1 테스트 가상 호스트 1 2015/01/22 100K / unlimit 200K 200K 200K 200K 2K/s active
2 테스트 가상 호스트 2 2015/01/22 100K / unlimit 200K 200K 200K 200K 2K/s active
Success

가상 호스트(Virtual host)와 관리자 계정이 정상적으로 생성 되었습니다.


* Host 이름을 클릭하여 설정을 변경 할 수 있습니다.

* account 컬럼 항목을 클릭하여 사용자를 등록하세요.
